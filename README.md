## Installation
  ### インストール
  ``` bash
    git clone https://github.com/kazutotakeuchi-32/slim_tutorial_app3.git
  ```
## Usage

  ### ローカルサーバ構築
  ```bash
    #初回
    composer install
    composer start
    php -S localhost:8080 -t  ./src/public
  ```
